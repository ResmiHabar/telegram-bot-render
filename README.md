# telegram-bot-render
Bu basit bir Telegram botudur ve Render.com üzerinde host edilmektedir.

## Özellikler
- /start - Başlangıç mesajı
- /help - Yardım
- /echo [mesaj] - Mesajı tekrarlama
- /info - Kullanıcı bilgileri
- Normal mesajlara cevap verme

## Kurulum
1. Render.com'da Web Service oluştur
2. Environment variable olarak `TELEGRAM_BOT_TOKEN` ekle
3. Deploy et!
